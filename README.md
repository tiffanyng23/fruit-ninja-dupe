# Fruit Ninja Dupe
A game inspired by Fruit Ninja.

Objective: Create as many fruit salads as you can.

Three fruits will be randomnly selected to form the ingredients for a fruit salad. For 60 seconds, fruits will fall down the screen. Players must select the correct fruit to create the fruit salad. In order to create a complete fruit salad, you must collect one of each fruit required. If you click on the wrong fruit, you will lose one point, 

At the end of the game, the number of complete fruit salads subtracted by the number of times you clicked the wrong fruit, will form your total score.

Example:
Banana - 6, Apple - 3, Coconut - 5, Deductions - 1

Complete Fruit Salads: 3, Deductions: 1

Final Score = 2
